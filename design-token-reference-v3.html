The 3-way icon split (Big / Descriptive / Action) is already present in v2 under different names. v3 renames into the color.icon.* convention and adds nav-selected as a first-class member.

13 CTA context modes New

v3 introduces a "context" mode axis on cta.radius so the same CTA component picks form / card / wallet / landing radius based on where it's dropped.

ContextResolves toRationale
formradius.cta.form (3)Sits next to text inputs; matches input radius.
cardradius.cta.card (12)Inside a card; matches card radius for visual coherence.
walletradius.cta.wallet (100)Deposit / withdraw CTAs are pill-shaped on 6 of 9 brands.
landingradius.cta.landing (100)Marketing CTAs are pill-shaped to read as decisive action.
How it works in Figma: the CTA component gets a "Context" variant property. Each variant binds cta.radius to a different context token. When a designer drops a CTA into a form frame, they select the "form" variant; the radius follows.
