# Design Tokens

ComeOn design token architecture — a 5-collection, multi-axis Figma Variables system covering brand (20 brands), theme (light/dark), jurisdiction (7 regulatory modes), semantic intent, and component overrides. Everything sits on a 4-point scale linked to a 16px body.
